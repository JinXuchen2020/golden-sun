# 2026-06-15 工作日志

## Phase 2 优化：GPU NPC 渲染 + 测试覆盖

### P0: NPC 渲染从 CPU 逐像素 blit 改为 GPU 纹理覆盖

**改动前**: NPC 精灵存储为 `Vec<Vec<u8>>`，每帧三重循环逐像素 alpha blend 写入 Mode 7 缓冲区。
**改动后**: NPC 精灵上传为 `Vec<Texture2D>`（与玩家一致），用 `draw_texture_ex` 叠加在 Mode 7 画面上，alpha blend 由 GPU 处理。

文件变更:
- `game/mod.rs`: `npc_sprites: [Vec<Vec<u8>>; 8]` → `npc: [Vec<Texture2D>; 8]`，`npc_frame()` → `npc_tex()`
- `game/draw.rs`: 移除 CPU blit 三重循环（~50 行），替换为 `render_npcs()` 方法（GPU draw_texture_ex）

### P1: 为 facing 函数添加 8 个测试

`facing_from_angle()` 和 `dir_from_delta()` 的 8 个单元测试覆盖了 8 象限分界和各主轴方向。

新增测试:
- `facing_from_angle_right/down/left/up_quadrant` — 4 方向象限边界
- `facing_from_angle_wraparound` — 360° 回绕
- `dir_from_delta_pure_horizontal/vertical` — 纯水平/垂直
- `dir_from_delta_dominant_axis` — 主方向判断

### P2: 重构 draw_world_map

提取 `render_npcs()` 方法（约 50 行），使 `draw_world_map()` 从 ~80 行缩减到 ~20 行。
